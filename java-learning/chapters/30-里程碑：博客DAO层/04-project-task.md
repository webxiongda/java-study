# Chapter 30 里程碑：博客DAO层 - 项目任务

## 任务概述

本章项目任务是：**交付博客数据访问层 v1**。任务必须服务于最终目标：能独立交付 Spring Boot REST API，并能在面试中讲清设计和实现。

## 业务背景

博客系统需要逐步具备数据访问、接口设计、认证授权、缓存、安全、测试、部署和面试包装能力。本章负责补齐其中的 **里程碑：博客DAO层** 能力，不能只停留在知识点阅读。

## 任务拆解

1. 阅读本章理论文档，提取 JDBC/MyBatis/SQL 整合、README、测试 的关键点。
2. 实现或设计“交付博客数据访问层 v1”的最小闭环。
3. 补充边界处理：非法输入、空数据、重复操作、依赖失败或并发冲突。
4. 用测试、日志、SQL、HTTP 请求或截图证明结果。
5. 写下你会如何向面试官介绍这部分实现。
