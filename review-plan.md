# 复习计划

复习节奏参考间隔重复：首次复习 +3 天，二次复习 +7 天，三次复习 +30 天。每次复习只做三件事：重跑代码、回答复盘题、更新错题。

## 自动化（需 Dataview）

> 在 `01-theory.md` frontmatter 加 `finished: 2026-05-20`，下面表格按完成日自动算 +3/+7/+30 复习日期。

### 今天该复习什么

````dataview
TABLE WITHOUT ID
  file.link as 章节,
  finished as 完成日,
  dateformat(date(finished) + dur(3 days), "yyyy-MM-dd") as 一刷,
  dateformat(date(finished) + dur(7 days), "yyyy-MM-dd") as 二刷,
  dateformat(date(finished) + dur(30 days), "yyyy-MM-dd") as 三刷
FROM "chapters"
WHERE file.name = "01-theory" AND finished
WHERE (
  date(finished) + dur(3 days)  = date(today)
  OR date(finished) + dur(7 days)  = date(today)
  OR date(finished) + dur(30 days) = date(today)
)
SORT chapter ASC
````

### 错题集

````dataview
TABLE WITHOUT ID
  date as 日期,
  chapter as 章节,
  symptom as 现象,
  status as 状态
FROM "mistakes"
SORT date DESC
````

> 上面 `mistakes` 查询需要把 `mistakes.md` 中每条错题改成单独的 `mistakes/yyyy-mm-dd-xxx.md` 文件，每条带 frontmatter（详见 `mistakes.md`）。
